# 🚀 Claude Code 新手入门指南

一份面向大学生的 Claude Code CLI 入门指南，用像教小学生一样有耐心的口吻，从零开始讲解如何用 Claude Code 写作业、做项目、画图表。

## 内容包含

- Terminal（终端）基础命令速查
- Claude Code 安装与启动
- 文件引用、键盘快捷键、常用指令
- LaTeX 作业编译（texpage.com）
- 交互式 HTML 演示文档生成
- Python 数据图绘制
- Plan Mode 说明
- Markdown 文件查看方式
- GitHub 仓库 / Skills 安装

## 文件

| 文件 | 说明 |
|------|------|
| `claude-code-beginner-guide.md` | Markdown 原版（可用 VS Code / Typora 查看） |
| `claude-code-beginner-guide.html` | 自包含 HTML 版（双击浏览器打开，支持深色模式） |
