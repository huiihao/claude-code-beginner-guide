# 🚀 Claude Code Beginner's Guide / 新手入门指南

A beginner's guide to Claude Code CLI for college students, written with the patience of teaching a child — from zero to getting things done: homework, projects, data visualization, and more.

一份面向大学生的 Claude Code CLI 入门指南，用像教小学生一样有耐心的口吻，从零开始讲解如何用 Claude Code 写作业、做项目、画图表。

## What's Inside / 内容包含

- Terminal basics / 终端基础命令速查
- Claude Code installation & startup / 安装与启动
- File referencing, keyboard shortcuts, slash commands / 文件引用、键盘快捷键、常用指令
- LaTeX assignment compilation (texpage.com) / LaTeX 作业编译
- Interactive HTML presentation generation / 交互式 HTML 演示文档生成
- Python data visualization / Python 数据图绘制
- Plan Mode explained / Plan Mode 说明
- Markdown file viewing / Markdown 文件查看方式
- GitHub repo & Skills installation / GitHub 仓库与 Skills 安装
